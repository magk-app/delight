## Tech Stack

Delight is built with modern, production-ready technologies:

### Frontend
- **Next.js 15** + React 19 + TypeScript + Tailwind CSS
- Server Components and App Router architecture
- Real-time updates via Server-Sent Events

### Backend
- **FastAPI** + Python 3.11+
- **SQLAlchemy 2.0** (async) + Alembic migrations
- **LangGraph** AI agents for multi-character system

### Infrastructure
- **Supabase** - Managed PostgreSQL with pgvector
- **Clerk** - Authentication and user management
- **OpenAI** - GPT-4o-mini for chat, GPT-4o for narrative
- **ARQ** - Async background job processing
