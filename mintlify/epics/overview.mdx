# Delight Epics Overview

Delight is structured around **8 interconnected epics** that build the complete emotionally intelligent AI companion platform. Each epic is designed to work together, creating a seamless user experience that combines AI coaching, narrative world-building, and structured progress tracking.

## Key Architecture Principles

These principles guide all epics:

## Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: FastAPI, SQLAlchemy 2.0, Pydantic, Alembic
- **Database**: Supabase (PostgreSQL + pgvector)
- **AI/LLM**: OpenAI GPT-4o-mini (chat), GPT-4o (narrative), text-embedding-3-small
- **Infrastructure**: ARQ (async jobs), Redis, Clerk (auth)
